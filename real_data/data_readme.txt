There are two folder of data, namely gut and soil. In each folder there are two subfolders namely raw_data and clean_data. The files in raw_data are directly coming from MG-RAST server while clean_data are processed. The human gut dataset is from https://www.mg-rast.org/mgmain.html?mgpage=project&project=mgp154, while the soil dataset is from https://www.mg-rast.org/mgmain.html?mgpage=project&project=mgp2592.
